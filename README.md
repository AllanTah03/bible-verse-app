# Bible Verse App

Application mobile Android de versets bibliques quotidiens — Bible de Jérusalem.

## Fonctionnalités

- Verset aléatoire du jour
- Fonctionne entièrement hors ligne
- Espace personnel pour ajouter ses propres versets

## Prérequis

- [Flutter SDK](https://flutter.dev/docs/get-started/install/windows) ≥ 3.0.0
- Android SDK (via Android Studio ou VS Code + extension Flutter)
- Android 7.0+ (API 24)

## Installation

```bash
flutter pub get
flutter run
```
